# vue-element-template

## 后台管理系统模板，参照大佬的项目做的
## 写得很拉胯，请勿用于上线项目，避免造出不必要的损失

## 技术栈：vue、element、mockjs、axios、svg、js-cookie、nprogress

## 项目开始

### 安装依赖
```
npm install
```

### 运行
```
npm run serve
```

### 打包
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
